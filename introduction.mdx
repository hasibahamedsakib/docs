# Project Documentation

## Overview

This project is a Next.js application with various features and functionalities, including user authentication, product management, order tracking, and more. The project uses several libraries and frameworks such as Chakra UI for styling, Redux for state management, and `next-i18next` for internationalization.

## Installation

To install the project dependencies, run:

#### Only Authentic Github User Can clone and run this project

```bash
git clone https://github.com/alecmithun/frontendevercel
cd frontendevercel
npm install
```
